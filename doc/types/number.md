# 数值

## 概述

### 整数和浮点数

JavaScript 内部，所有数字都是以64位浮点数形式储存，整数也是如此。所以，`1`与`1.0`是相同的，是同一个数

```javascript
1 === 1.0 // true
 ```

 底层没有整数，所有的数字都是小数（64位浮点数）。容易造成混淆的是，某些运算只有整数才能完成，此时 JavaScript 会自动把64位浮点数，转为32位整数，然后再进行运算。

 由于浮点数不是精确的值，所以涉及小数的比较和运算要特别小心。

 ```javascript
0.1 + 0.2 === 0.3
// false

0.3 / 0.1
// 2.9999999999999996

(0.3 - 0.2) === (0.2 - 0.1)
// false
```
